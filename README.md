# sistemmanajemendisposisi
Aplikasi web internal Disposisi Surat yang digunakan khusus di lingkungan Bagian Layanan dan Pembinaan Sumber Daya Manusia, Biro Sumber Daya Manusia Aparatur, Organisasi dan Ketatalaksanaan, Sekretariat Jenderal, Kementerian Imigrasi dan Pemasyarakatan

Aplikasi ini bertujuan untuk mendukung proses penerimaan, pendistribusian, pengerjaan, pemantauan, hingga penyelesaian disposisi surat secara tertib, terdokumentasi, dan mudah diaudit.

Aplikasi digunakan oleh Administrator, Ketua Tim, dan Anggota dalam lingkup bagian tersebut, dengan alur kerja disposisi yang jelas, terdokumentasi, dan berbasis peran.

Seluruh data aplikasi bersumber dari database utama dan dikelola secara terpusat oleh Bagian Layanan dan Pembinaan Sumber Daya Manusia Aparatur, Organisasi dan Ketatalaksanaan.
